Updating Thirdlane PBXÔÉÅ

Once installed, Thirdlane provides their own update repositories for their pbx software. You can update an install as seen below

yum clean all
yum update

This will include release upgrades. You can see what resale versions this will bring you to by looking at the pbxmw-mt package. For example, below we will refresh all repositories and then look for the next installation candidate. In this example we want to update from Thirdlane version 14.3.1 to 15.1.2.

yum clean all
yum make
yum list pbxmw-mt --showduplicates
