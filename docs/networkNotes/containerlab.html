Containerlab notes

Bring up a lab

containerlab deploy

Bringing down a lab

containerlab destroy

Restarting an individual container

dcoker restart [container name]

Managing images

All images are docker images. You can mnnage them as below.
Listing images
root@r620:~# docker images
REPOSITORY                            TAG        IMAGE ID       CREATED          SIZE
vrnetlab/cisco_vios                   15.6.7     c2639dfa6e6d   37 minutes ago   545MB

Removing images

docker rmi c2639dfa6e6d

Saving an image

docker save c2639dfa6e6d | gzip > docker_cisco_vios_15_6_7.tar.gz

Cisco vios

Username: admin
Password: admin
cisco.clab.yaml
name: cisco

topology:
  nodes:
    vios:
      kind: linux
      image: vrnetlab/cisco_vios:15.6.7
      env:
        HOSTNAME: vios

  links:
    - endpoints: ["vios:eth1"]
Documentation:
containerlab vios
vrnetlab vios
